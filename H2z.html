Command line basics revisited

  • CLI oli olemassa ennen Windowsia, Linuxia tai internettiä
  • $ -merkki tarkoittaa normaali käyttäjää
  • # -merkkillä voidaan lisätä "kommentti"
  • sudo -komento antaa root oikeudet
  • man -komento avaa ohjelman manuaalisivun josta löytyy kaikki tarvittava ohjelmaa koskien. Samankaltainen, mutta vähemmän tietoa antava komento --help

Komentoja voidaan ketjuttaa

dsq

  • CLI työkalu, jolla voidaan suorittaa SQL kyselyitä muun muassa csv, JSON ja excel tidostoihin
  • Muita samankaltaisia työkaluja q, textgl ja octosql
